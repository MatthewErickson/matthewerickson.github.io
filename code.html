Find Max Value

Problem

Given: array of integers
Return: the maximum value in the array

Solution

Just loop through and keep track of the biggest number you've seen.

Code

public int findMaxValue(int[] array) {
    int max = Integer.MIN_VALUE;
    for (int num : array) {
        max = Math.max(max, num);
    }
    return max;
}

Union-Find

Problem

Given: N/A
Return: N/A

Solution

Union-Find is a neat algorithm for joining forests in graphs optimally. For a more in-depth explanation, see this document.

Code

public class UnionFind {
        private int[] id;
        private int n;

        public UnionFind(int n) {
            this.n = n;
            id = new int[n];
            for (int i = 0; i < n; i++) {
                id[i] = i;
            }
        }

        public void union(int p, int q) {
            id[root(p)] = root(q);
        }

        public boolean find(int p, int q) {
            if (0 <= p && p < n && 0 <= q && q < n) {
                return root(p) == root(q);
            }
            return false;
        }

        private int root(int i) {
            while (i != id[i]) {
                id[i] = id[id[i]];
                i = id[i];
            }
            return i;
        }
    }
